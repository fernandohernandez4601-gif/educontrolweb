# EduControl Web

Sistema web desarrollado en React para el control académico básico.

## Objetivo
Desarrollar un sistema que permita gestionar alumnos y controlar información académica.

## Metodología
El proyecto se desarrolla utilizando SCRUM, organizando el trabajo en sprints.

## Sprint 1
- Inicio de sesión
- Registro de alumnos
- Listado de alumnos

## Tecnologías
- React
- Node.js
- JavaScript

## Instalación
1. Clonar repositorio
2. Ejecutar npm install
3. Ejecutar npm start
